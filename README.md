The objective of this project is to be able to study Spring Framework with something concrete building a backend to make financial control.

## Prerequisites

* Maven
* Java JDK 11+
* Postgres

## Running

Just run the maven goal

```
docker-compose up -d
mvn spring-boot:run
```
